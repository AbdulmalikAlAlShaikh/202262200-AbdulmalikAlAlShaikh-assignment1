## âœ¨ Key Features Explained

### 1. Responsive Design

The website adapts seamlessly to different screen sizes using:

- CSS Grid and Flexbox layouts
- Mobile-first approach with breakpoints at 480px, 768px, and 968px
- Responsive navigation with mobile menu

### 2. Theme Switching

Users can toggle between dark and light themes:

- Preference saved in localStorage
- Respects system preference on first visit
- Smooth transitions between themes

### 3. Interactive Elements

- **Time-based Greeting**: Dynamic greeting based on time of day
- **Form Validation**: Real-time validation with helpful error messages
- **Scroll Animations**: Elements fade in as you scroll
- **Mobile Menu**: Hamburger menu for mobile devices
